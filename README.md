# Website-for-the-Moscow-pre-professional-Olympiad.-2025-case-4

Веб-приложение для обучения компьютерной грамотности с использованием скриншотов и голосового сопровождения.

## О проекте

Это адаптивный веб-сайт, предназначенный для обучения пользователей работе с компьютерными программами и интерфейсами через визуальные примеры. Приложение демонстрирует скриншоты с подробными описаниями и предлагает дополнительные обучающие функции.

## Основные возможности

- **Адаптивный интерфейс** - корректное отображение на ПК, планшетах и смартфонах
- **Два уровня сложности** - базовый и расширенный режимы обучения
- **Голосовое сопровождение** - озвучивание инструкций и описаний
- **Визуальные подсказки**:
  - Анимированный курсор для указания важных элементов
  - Подсветка активных зон интерфейса
  - Интеграция скриншотов с детальными пояснениями
- **Упрощенная навигация** - линейное прохождение с кнопками "Назад"/"Далее"
- **Интерактивная помощь** - всплывающее окно с инструкциями по использованию
- **Индикатор прогресса** - визуальное отображение пройденного материала

## Требования к данным

### Формат файлов со скриншотами
- **Изображения**: JPG, PNG или WebP форматы
- **Текстовые описания**: TXT файлы с UTF-8 кодировкой

### Именование файлов
Файлы должны быть названы соответствующим образом:
- `screenshot1.jpg` + `screenshot1.txt`
- `screenshot2.jpg` + `screenshot2.txt`
- и т.д.

### Формат текстовых файлов
Первая строка - заголовок скриншота, остальной текст - подробное описание.

Пример `screenshot1.txt`:
```
Работа с текстовым редактором
На этом скриншоте показан интерфейс текстового редактора. Основные элементы: панель инструментов с кнопками форматирования, область ввода текста и линейка для установки отступов. Обратите внимание на кнопки сохранения и печати в верхнем левом углу.
```

## Настройка проекта

### 1. Добавление собственных скриншотов
1. Поместите изображения скриншотов в папку `src/`
2. Создайте соответствующие txt-файлы с описаниями
3. Обновите массив `screenshotsData` в файле `index.html`:

```javascript
const screenshotsData = {
    basic: [
        {
            image: "src/ваш_файл1.jpg",
            title: "Ваш заголовок",
            description: "Ваше описание..."
        },
        // ... другие скриншоты
    ],
    advanced: [
        // ... скриншоты для продвинутого уровня
    ]
};
```

### 2. Настройка голосового сопровождения
Приложение использует Web Speech API. Для корректной работы:
- Убедитесь, что браузер поддерживает синтез речи
- Проверьте наличие русскоязычных голосов в системе
- При необходимости измените параметры речи в функции `initializeSpeech()`

## Кастомизация

### Изменение цветовой схемы
Отредактируйте CSS-переменные в секции `:root`:

```css
:root {
    --primary: #4a6fa5;      /* Основной цвет */
    --primary-dark: #385d8a; /* Темный оттенок */
    --secondary: #f0f4f8;    /* Вторичный цвет */
    --accent: #ff9e4a;       /* Акцентный цвет */
    --text: #333;            /* Цвет текста */
    --bg: #fff;              /* Фон */
}
```

### Добавление новых уровней сложности
1. Добавьте новый ключ в объект `screenshotsData`
2. Создайте соответствующую кнопку в HTML
3. Добавьте обработчик события для переключения
